# External Integrations

**Analysis Date:** 2026-04-01

## APIs & External Services

**Social/Profile:**
- GitHub - Author social link; configured as `params.social.GitHub = "jcbmcn"` in `hugo.toml`. Rendered by LoveIt theme as a social icon link.
- LinkedIn - Author social link; configured as `params.social.Linkedin = "jacob-mcneilly-a6a516112"` in `hugo.toml`. Rendered by LoveIt theme.

**Ignored/Stubbed:**
- Instagram Access Token - Explicitly ignored via `ignoreErrors = ["error-missing-instagram-accesstoken"]` in `hugo.toml`. Instagram integration is not active.
- Remote JSON - Explicitly ignored via `ignoreErrors = ["error-remote-getjson"]` in `hugo.toml`. No active remote JSON endpoints.

**Error Tracking:**

## CI/CD & Deployment

**Hosting:**
- GitHub Pages - Static site hosting at `https://jcbmcn.com/`
  - Deployed via `actions/deploy-pages@v4`
  - Build artifact path: `./public`

**CI Pipeline:**
- GitHub Actions - Workflow defined at `.github/workflows/hugo.yml`
  - Trigger: Push to `main` branch or manual `workflow_dispatch`
  - Runner: `ubuntu-latest`
  - Steps: Install Hugo Extended 0.128.0, Install Dart Sass, Checkout (with submodules recursive), Configure Pages, Build with `hugo --minify`, Upload artifact, Deploy
  - Permissions: `contents: read`, `pages: write`, `id-token: write`
  - Concurrency group: `"pages"` (prevents overlapping deployments, does not cancel in-progress runs)

**GitHub Actions used:**
- `actions/checkout@v4`
- `actions/configure-pages@v5`
- `actions/upload-pages-artifact@v3`
- `actions/deploy-pages@v4`

## Environment Configuration

**Required env vars:**
- None required at runtime. The site is fully static.
- CI uses `HUGO_VERSION: 0.128.0` (defined in workflow env) and `HUGO_ENVIRONMENT: production` (set at build step).
- `HUGO_CACHEDIR` set to `${{ runner.temp }}/hugo_cache` during CI build.

**Secrets location:**
- No application secrets. GitHub Pages deployment uses the auto-provided `GITHUB_TOKEN` with scoped permissions defined in the workflow.

## Webhooks & Callbacks

**Incoming:**
- None.

**Outgoing:**
- None.

## SEO & Metadata

**Open Graph / Twitter Cards:**
- Configured natively via Hugo/LoveIt theme using `params.title`, `params.description`, and `params.images = ["/logo.png"]` in `hugo.toml`.

**Sitemap:**
- Auto-generated by Hugo at `sitemap.xml`; configured under `[sitemap]` in `hugo.toml` with `changefreq = "weekly"` and `priority = 0.5`.

**RSS:**
- Auto-generated by Hugo/LoveIt theme. Enabled via `params.social.RSS = true` in `hugo.toml`.

**robots.txt:**
- Auto-generated by Hugo (`enableRobotsTXT = true` in `hugo.toml`).
