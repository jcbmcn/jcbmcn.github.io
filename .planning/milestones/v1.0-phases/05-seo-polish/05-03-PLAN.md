**Change 2 — Add anti-flash inline script as FIRST element in `<head>`:**
Insert immediately after the opening `<head>` tag, before `<meta charset>`:
```astro
<script is:inline>
  (function() {
    const stored = localStorage.getItem('theme');
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    const theme = stored ?? (prefersDark ? 'dark' : 'light');
    document.documentElement.setAttribute('data-theme', theme);
  })();
</script>
```

**WHY first:** The anti-flash script must execute before the browser parses any CSS. If placed after `<link rel="stylesheet">` or `<style>`, the browser may render one frame with the wrong theme. `is:inline` prevents Astro from bundling or deferring it.

**Change 3 — Add RSS autodiscovery link in `<head>`** (after `<meta name="description">` line):
