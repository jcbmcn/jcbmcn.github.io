  <action>
    Make three changes to `src/components/PostCard.astro`:

    **1. Add `data-tags` attribute to `<article>`:**
    Change:
    ```astro
    <article class="post-card">
    ```
    To:
    ```astro
    <article class="post-card" data-tags={tags.join(',')}>
    ```
    This allows the filter script to show/hide cards by tag match.

    **2. Change tag chips from `<li>` to `<li><button>` (keep list wrapper for valid HTML):**
    Change:
    ```astro
    {tags.map((tag) => (
      <li class="post-card__tag">{tag}</li>
    ))}
    ```
    To:
    ```astro
    {tags.map((tag) => (
      <li>
        <button type="button" class="post-card__tag" data-tag={tag} aria-pressed="false">
          {tag}
        </button>
      </li>
    ))}
    ```
    Note: The `<li>` wrapper is kept because `<button>` is not valid as a direct child of `<ul>`. The `post-card__tag` class moves to the button.

    **3. Update `<ul class="post-card__tags">` list-style and add active button styles to the `<style>` block:**

    The `.post-card__tag` rule already exists (targeting `<li>`). Update it to target `button.post-card__tag` and add the active state class. Add to the scoped `<style>` block:

    ```css
    /* Update existing .post-card__tags ul to remove list item gaps from <li> wrappers */
    .post-card__tags li {
      display: contents; /* removes <li> box so only the button is visible */
    }

    /* Move existing .post-card__tag styles to button element */
    /* (The .post-card__tag rule already has the correct styles — just ensure it applies to <button>) */
    .post-card__tag {
      /* existing styles remain: font-size, font-weight, color, background, padding, border-radius */
      border: none;        /* remove default button border */
      cursor: pointer;     /* pointer cursor per UI-SPEC */
      font-family: inherit; /* match body font */
      line-height: 1;
    }

    .post-card__tag:hover {
      background: var(--color-accent);
      color: white;
    }

    .post-card__tag--active {
      background: var(--color-accent);
      color: white;
    }
    ```

    **Do NOT change** the `.post-card__tags` flex/gap/wrap rules — those remain identical. Only add the `li { display: contents }`, cursor/border overrides on the button, and the `--active` class.

    Per CONTEXT.md D-07: Tags on post cards toggle the same filter as the filter bar chips. The click handler is wired in Plan 02's `<script>` block — this task only adds the markup and CSS needed.
  </action>
